# terminal_http_api

------



## API

| 接口    | 类型 | 参数                          | 返回      | 说明                                   |
| ------- | ---- | ----------------------------- | --------- | -------------------------------------- |
| /all    | GET  | 无                            | [1,2,3,5] | 显示所有运行中的终端的ID               |
| /create | GET  | 无                            | 34        | 创建一个新的终端                       |
| /close  | GET  | id=34                         | 0或1      | 关闭终端, 成功则返回1                  |
| /run    | GET  | cmd="ping 8.8.8.8"<br />id=34 | 0或1      | 指定ID的终端运行命令, 成功则返回1      |
| /result | GET  | id=34                         | 字符串    | 指定ID的终端最后一次显示的命令运行结果 |
